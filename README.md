# Stock Insight Dashboard

주식 데이터를 분석하고 인사이트를 도출하는 대시보드 프로젝트입니다.

## 프로젝트 개요

다양한 주식 데이터를 수집·시각화하고, 기술적 지표와 통계 분석을 통해 투자 인사이트를 제공하는 웹 대시보드입니다.

## 주요 기능

- 실시간 및 과거 주가 데이터 조회
- 기술적 지표 시각화 (이동평균선, RSI, MACD 등)
- 종목별 인사이트 리포트
- 포트폴리오 수익률 분석
- 시장 트렌드 및 섹터별 동향 파악

## 기술 스택

> 기술 스택은 개발 진행에 따라 업데이트될 예정입니다.

## 시작하기

```bash
# 저장소 클론
git clone https://github.com/JaeYong0107/stock-insight.git
cd stock-insight

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
