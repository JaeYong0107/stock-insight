## 3. Epic 목록

## Epic 1. 프로젝트 기반 구조 및 데이터 모델 구축

### Epic 제목

`[EPIC] 프로젝트 기본 구조 및 DB 모델 구축`

### Epic 설명

Stock Insight 백엔드의 기본 구조를 세팅하고, MVP 핵심 도메인에 대한 SQLAlchemy 모델과 Alembic 마이그레이션을 구성한다.

### 포함 범위

- FastAPI 프로젝트 구조 정리
- DB 세션/엔진 구성
- 핵심 모델 구현
- 초기 마이그레이션 작성

### 완료 기준

- 백엔드 프로젝트 구조가 정리되어 있다.
- 핵심 테이블 모델이 구현되어 있다.
- Alembic 마이그레이션이 실행 가능하다.

## 4. Story 및 Task 상세

## Epic 1 하위 Story

### Story 1-1

#### 제목

`[STORY] 백엔드 기본 구조와 공통 인프라를 구성한다`

#### 설명

FastAPI, 설정 로딩, DB 연결, 기본 디렉터리 구조를 정리해 이후 구현의 기반을 만든다.

#### 완료 기준

- 공통 구조가 정리되어 있다.
- DB 연결 구성이 준비되어 있다.

#### Tasks

`[TASK] FastAPI 라우터 및 디렉터리 구조 정리`

- 설명: `api`, `schemas`, `models`, `services`, `repositories`, `tasks` 구조를 정리한다.
- 완료 기준: 각 디렉터리와 기본 모듈 구조가 생성되어 있다.

`[TASK] 설정 및 환경 변수 로딩 구조 구현`

- 설명: Pydantic settings 기반으로 환경 변수 로딩 구조를 정리한다.
- 완료 기준: DB, Redis, AI provider 설정이 일관되게 로드된다.

`[TASK] SQLAlchemy 엔진 및 세션 구성`

- 설명: async SQLAlchemy 엔진과 세션 팩토리를 구성한다.
- 완료 기준: 앱 전역에서 재사용 가능한 DB 세션 구성이 준비된다.

### Story 1-2

#### 제목

`[STORY] MVP 핵심 데이터 모델과 초기 마이그레이션을 구현한다`

#### 설명

설계 문서 기준 핵심 테이블을 SQLAlchemy와 Alembic으로 구현한다.

#### 완료 기준

- 핵심 모델이 구현되어 있다.
- Alembic 마이그레이션이 적용 가능하다.

#### Tasks

`[TASK] stocks 및 stock_price_daily 모델 구현`

- 설명: 종목과 일봉 가격 모델을 구현한다.
- 완료 기준: FK, unique, index가 반영되어 있다.

`[TASK] 뉴스 관련 모델 구현`

- 설명: `news_articles`, `news_article_stocks`, `news_sentiment_runs` 모델을 구현한다.
- 완료 기준: 뉴스-종목 연결 및 감성 이력 구조가 반영되어 있다.

`[TASK] 인사이트 및 평가 모델 구현`

- 설명: `stock_insights`, `stock_insight_evaluations` 모델을 구현한다.
- 완료 기준: 인사이트 스냅샷과 평가 구조가 반영되어 있다.

`[TASK] 작업 실행 이력 모델 구현`

- 설명: `collection_jobs`, `collection_runs` 모델을 구현한다.
- 완료 기준: 작업 상태 추적에 필요한 컬럼과 인덱스가 반영되어 있다.

`[TASK] Alembic 초기 마이그레이션 작성`

- 설명: 위 모델을 기준으로 초기 마이그레이션을 생성한다.
- 완료 기준: `alembic upgrade head`가 정상 수행된다.
